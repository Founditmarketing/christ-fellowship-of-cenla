# Christ Fellowship of Cenla

A Reformed Baptist church in Central Louisiana devoted to the fellowship of the gospel and the 1689 London Baptist Confession.

**Live site**: [cfcenla.com](https://cfcenla.com)

## Tech Stack

- React 19 + TypeScript
- Vite
- TailwindCSS v4
- Framer Motion
- Lucide React icons

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
